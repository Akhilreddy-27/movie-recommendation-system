### 🕹️ How It Works (in the UI)
1. **Search** for a movie in the dropdown.
2. Click **Recommend**.
3. See the **Top 5 similar movies** computed using **cosine similarity** over preprocessed features.

---

### 🧪 Example Output

Search: **"The Dark Knight"**

2. **The Dark Knight Rises**
